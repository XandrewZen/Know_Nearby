# Know_Nearby

📍 Know Your Area

Know Your Area is a location-based marketplace app built with Python. It allows users to discover, upload, and sell products within their nearby area using a map-based interface.

🚀 Features
🗺️ Location-based product browsing (nearby marketplace concept)
📦 Users can upload products for sale
🔍 Browse and search items by area
💬 Simple and fast user interaction
🧠 Built using Python with Kivy frontend and Flask / Socket backend
🗄️ MongoDB database for storing users and products
🛠️ Tech Stack
Python 🐍
Kivy (UI framework)
Flask / Socket (backend communication)
MongoDB (database)
GPS / Map integration (future upgrade)
🎯 Goal

To create a hyper-local marketplace platform where people can easily buy and sell items in their nearby area without complex setup.

🔮 Future Improvements
Real-time chat between buyers and sellers
Map-based product pins
Payment integration
Mobile version support
Advanced filtering by distance & category
